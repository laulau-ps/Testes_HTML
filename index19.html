O elemento label define um rótulo para vários outros elementos. É útil para leitores de tela, ajuda os usúarios que tem dificuldade em clicar em regiões muito pequenas da tela. O atributo id do label deve ser igual ao id do input. Exemplo:

form action="">
label for="nome">Primeiro nome: label>
input type="text" name="nome" id="nome">
form>
